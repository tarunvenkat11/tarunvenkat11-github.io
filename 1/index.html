Project 1

The project was to create full color image from the Prokudin-Gorskii collection, which I accomplished by conducting an exhaustive search for the optimal alignment of the three plates with a specified window. For larger images, this process would have taken too long, so I implemented an recursive image pyramid approach with a scale factor of 0.5.
